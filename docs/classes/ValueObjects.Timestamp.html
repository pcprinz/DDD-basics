compare

  • compare(other: "now" | PlainDateTime, density?: "years" | "months" | "days" | "hours" | "minutes" | "seconds" | "milliseconds"): 0 | 1 | -1

  • compares this date with a given other PlainDateTime or 'now' to compare it with the current dateTime.

    • returns -1 if this dateTime is earlier than the other one
    • returns 0 if the dates are equal
    • returns 1 if this dateTime is later than the other one
    Examples
     
    '2020-05-25'.compare('2020-09-18') => -1
    '2020-05-25'.compare('2020-01-01') => 1
    '2020-05-25'.compare('2020-05-06', 'months') => 0

    Returns

    -1 | 0 | 1 indicating which dateTime is more recent

    Parameters

    • other: "now" | PlainDateTime

      the PlainDateTime to compare to / 'now' to use the current dateTime

    • density: "years" | "months" | "days" | "hours" | "minutes" | "seconds" | "milliseconds" = 'milliseconds'

      the density the comparison has to have

    Returns 0 | 1 | -1

distance

  • distance(toOther: "now" | PlainDateTime, density?: "years" | "months" | "days" | "hours" | "minutes" | "seconds" | "milliseconds"): number

  • compares this date with a given other PlainDateTime or 'now' to compare it with the current dateTime and returns the distance between both.

    • distance = other - this
      • positive result = other dateTime was later
      • negative result = other dateTime was earlier
    • units depending on density:
      • "years" | "months" | "days" = days
      • all other are fitting

    Returns

    the distance between the dates

    Parameters

    • toOther: "now" | PlainDateTime
    • density: "years" | "months" | "days" | "hours" | "minutes" | "seconds" | "milliseconds" = 'milliseconds'

      the density the comparison has to have

    Returns number
